# 快速开始指南

## Windows 用户（推荐）

### 一键安装
```
双击 scripts\install.bat
```

### 一键启动
```
双击 scripts\start.bat
```

---

## macOS / Linux 用户

### 一键安装
```bash
chmod +x scripts/*.sh
./scripts/install.sh
```

### 一键启动
```bash
./scripts/start.sh
```

---

## 虚拟环境说明

安装脚本会自动在 `backend/venv` 目录创建 Python 虚拟环境。

**重装系统后恢复步骤：**
1. 安装 Python 3.10+ 和 Node.js 18+
2. 重新运行安装脚本
3. 音色文件和模型文件如果之前备份了可直接复制回来

**手动激活虚拟环境：**
```bash
# Windows
cd backend
venv\Scripts\activate

# macOS/Linux
cd backend
source venv/bin/activate
```

## 常见问题

**Q: 重装系统后需要重新下载模型吗？**
A: 如果备份了 `backend/checkpoints` 目录，直接复制回来即可，无需重新下载。

**Q: 虚拟环境损坏怎么办？**
A: 删除 `backend/venv` 目录，重新运行安装脚本。

**Q: 如何更新项目？**
A: 
```bash
git pull
# 然后重新运行安装脚本
```
